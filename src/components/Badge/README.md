# Badge

```html
<Badge
    pill
    variant="danger"
>
    I am a badge
</Badge>
```

## Props

| Name | Type | Description |
| - | - | - |
| pill | boolean | Makes the badge more rounded. |
| variant | [Variant](#link-to-variant-definition) (string) | The variant of the badge. |

## Slots

| Name | Description |
| - | - |
| default | Displays content inside the badge. |
