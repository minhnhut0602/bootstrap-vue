# Accordion

```vue
<Accordion>
    <AccordionItem header="Header">
        Content
    </AccordionItem>
</Accordion>
```

---

## Slots

| Name    | Description                           | Bindings                                               |
| ------- | ------------------------------------- | ------------------------------------------------------ |
| default | Displays the content of the accordion | **parent-id** `mixed` - The id of the parent accordion |
